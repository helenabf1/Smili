## Descrição
Smili é um rato que precisa atravessar a cidade desviando de carros, ratoeiras e do esgoto para coletar queijos. O jogo fica progressivamente mais difícil conforme a pontuação aumenta, com a câmera subindo cada vez mais rápido.

## Como instalar as dependências

Certifique-se de ter o Python instalado. Em seguida, instale o PyGame:
## pip install pygame

## Como rodar o jogo

Dentro da pasta `rato_na_rua`, execute:
## python main.py

## Controles
- **Setas ou WASD** — mover o Smili
- **P** — pausar o jogo
- **I** — tela de instruções
- **R ou ENTER** — jogar de novo após game over
- **ESC** — voltar ao menu
